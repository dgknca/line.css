# line.css

A simple CSS framework that provides useful components.

[Documentatin](https://dgknca.github.io/line.css/)

## Install

CDN

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/line.css@0.1.0/line.min.css"/>
```

Also you can install via npm

```js
npm i line.css
```

and import it

```js
import 'line.css' 
```

## Development

- `npm install`   
- `gulp`

For storybook:

- `npm run storybook`

For documentation:

- `cd docs`
- `npm install`
- `npm run serve`
